## Projects

This repository contains example projects I'm experimenting with. The newest addition is a small Tetris implementation using Python's `curses` module.

### Running the Tetris game

1. Make sure Python 3 is installed.
2. Run the script directly from the terminal:
   ```bash
   python3 tetris.py
   ```
   Use the arrow keys to move and rotate the pieces. Press any key after `GAME OVER` to exit.
